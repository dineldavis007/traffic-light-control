# 🚦 Traffic Light Control System

## 📌 Project Overview

This project is a Traffic Light Control System using Arduino UNO and Java Swing GUI.

The system allows users to control:
- Red Light
- Yellow Light
- Green Light
- Standard Checking Mode
- Buzzer Alerts

Communication between Java and Arduino is done using Serial Communication.

---

# 🛠 Components Used

- Arduino UNO
- Red LED
- Yellow LED
- Green LED
- Buzzer
- Breadboard
- Jumper Wires

---

# 💻 Software Used

- Arduino IDE
- IntelliJ IDEA

- jSerialComm Library

✅ Traffic light control

✅ Buzzer alerts  

---

# 🔌 Arduino Pins

| Component | Pin |
|---|---|
| Red LED | D3 |
| Yellow LED | D4 |
| Green LED | D5 |
| Buzzer | D6 |

---

# 🚀 Commands

| Command | Action |
|---|---|
| RED | Turn ON red light |
| YELLOW | Turn ON yellow light |
| GREEN | Turn ON green light |
| CHECK | Run checking sequence |
